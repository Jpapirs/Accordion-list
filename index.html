Roadmap.sh is a community effort to create learning paths, guides, project ideas and other similar content to help developers grow in their careers.

We plan to continue expanding our collection of roadmaps, introduce interactive features, support more technologies, and build tools that help developers and teams track their progress.
How is roadmap.sh built? arrow_drop_up
Roadmap.sh is an open-source project built using modern web technologies like React and Node.js. The content is community-driven and maintained through GitHub contributions.

Yes, roadmap.sh can be used by teams to guide learning, onboard new developers, and align on skill development. Custom roadmaps and progress tracking are particularly useful in team settings.
